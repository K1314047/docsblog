## EP003｜Windows 本地部署 openclaw（大模型实践）

- 视频教程：https://www.bilibili.com/video/BV1UgA6zUEpS/?share_source=copy_web
- 文字教程：https://share.note.youdao.com/s/dbvfMzXe
- API Key：https://openrouter.ai/settings/keys?utm_source=signup-success
- Telegram 代理配置

```
openclaw config set channels.telegram.proxy "http://127.0.0.1:7890"
```

> 📌 注意：将 `7890` 替换为你本地代理端口
